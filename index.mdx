## What is rshell?

rshell is a Go library that embeds a restricted POSIX-like shell interpreter designed specifically for AI agents. Unlike a general-purpose shell, rshell is **default-deny**: every resource — commands, filesystem paths, environment variables, and output redirections — is blocked unless explicitly permitted by the caller.

This means an AI agent running shell commands through rshell cannot:
- Execute commands not in an explicit allowlist
- Read or write files outside permitted directories
- Inherit sensitive environment variables from the host process
- Write to arbitrary files (only `/dev/null` is a valid redirect target)

## How it works

rshell wraps the [`mvdan.cc/sh`](https://pkg.go.dev/mvdan.cc/sh/v3) shell parser and replaces the execution engine with a sandboxed runner. Each `interp.New()` call creates an isolated `Runner` with your specified security policy:

```go main.go
package main

import (
    "context"
    "os"
    "strings"
    "time"

    "github.com/DataDog/rshell/interp"
    "mvdan.cc/sh/v3/syntax"
)

func main() {
    script := `
        files=$(find /data -name "*.log" -type f)
        wc -l $files
    `

    prog, _ := syntax.NewParser().Parse(strings.NewReader(script), "")

    runner, _ := interp.New(
        interp.StdIO(nil, os.Stdout, os.Stderr),
        interp.AllowedCommands([]string{"rshell:find", "rshell:wc"}),
        interp.AllowedPaths([]string{"/data"}),
        interp.MaxExecutionTime(30*time.Second),
    )
    defer runner.Close()

    runner.Run(context.Background(), prog)
}
```

## Security model at a glance

Every access path is **default-deny**:

| Resource | Default | How to enable |
|---|---|---|
| Command execution | All commands blocked (exit code 127) | `AllowedCommands([]string{"rshell:cat"})` or `AllowAllCommands()` |
| External commands | Blocked (exit code 127) | Provide an `ExecHandler` |
| Filesystem access | Blocked | `AllowedPaths([]string{"/my/dir"})` |
| Environment variables | Empty — no host env inherited | `Env("KEY=value")` |
| Output redirections | Only `/dev/null` allowed | N/A — by design |

<Note>
  rshell is designed to be used as a Go library. The `rshell` CLI binary is included for testing and scripting convenience, but the primary interface is the `interp` package.
</Note>
